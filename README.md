
# Book API

A simple REST API to manage a list of books with basic CRUD operations.

## Features

- Add a new book (POST)
- Get all books (GET)
- Get a book by ID (GET)
- Update a book by ID (PUT)
- Delete a book by ID (DELETE)

## Getting Started

### Prerequisites

- Node.js installed
